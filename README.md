# Rackrack

[![Build Status](https://travis-ci.org/okitan/rackrack.svg?branch=master)](https://travis-ci.org/okitan/rackrack)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rackrack'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rackrack
